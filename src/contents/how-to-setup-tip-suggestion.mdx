# Setup tip suggestion

### To show tip suggestion

1. Go to license web 
2. Choose **Store** tab
3. Choose your store
4. Hit **Credit Card** tab, choose **Kiosk** section
5. In **Tip** section, check **Enable Tip Suggestion**
6. Hit **Save** button


### To hide button no tip

1. In **Tip** section, check **Hide button No Tip On Tip screen**
2. Hit **Save** button

### To change tip suggestion Mode

1. In **Tip** section, choose **Tip Suggestion Mode** (Amount/Percent)
2. Hit **Save** button
