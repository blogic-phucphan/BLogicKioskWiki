# Show print button in confirm screen

To show print button in confirm screen

1. Go to license web 
2. Choose **Store** tab
3. Choose your store
4. Hit **Credit Card** tab, choose **Kiosk** section
5. In **Configurations** section, check **Show Print Receipt**
6. Hit **Save** button
