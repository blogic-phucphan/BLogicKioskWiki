# Show Print Button On The Confirmation Screen

To show the print button on the confirmation screen:

1. Go to the license website.
2. Choose the **Store** tab.
3. Select your store.
4. Go to the **Credit Card** tab and choose the **Kiosk** section.
5. In the **Configurations** section, enable the **Show Print Receipt** option.
6. Click the **Save** button.
