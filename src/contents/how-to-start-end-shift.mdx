# Start/End Shift

To start a shift:

1. Go to **Settings** (Click 5 times on the footer in the start screen).
2. Click on **Start shift**.

To end a shift:

1. Go to **Settings** (Click 5 times on the footer in the start screen).
2. Click on **End shift**.
