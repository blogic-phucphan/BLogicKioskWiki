# Change Workstation Name

To change workstation name:

1. Go to **Setting** (Click 5 times on the footer in the start screen)
2. Choose **Settings** section
3. Hit **Devices** tab
4. Hit **Workstation Name**
5. Edit name
6. Hit **Save**

<Blockquote type="warning">
  Workstation name will be changed in pos dashboard
</Blockquote>
