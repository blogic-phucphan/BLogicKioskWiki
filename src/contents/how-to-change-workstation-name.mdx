# Change Workstation Name

To change the workstation name, follow these steps:

1. Go to **Setting** (Click 5 times on the footer in the start screen).
2. Choose the **Settings** button
3. Click on the **Devices** tab
4. Select **Workstation Name**
5. Edit the name
6. Click **Save**

<Blockquote type="warning">
 The workstation name will be changed in the POS dashboard.
</Blockquote>
