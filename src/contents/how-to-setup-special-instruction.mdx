# Setup special instructions for item, ticket

To Setup special instructions for item, ticket

1. Open pos dashboard
2. Hit **Setting** tab
3. Choose **Configurations**
4. Hit **Kiosk** tab
5. Check **Allow Special Instruction for ticket** (ticket), Check **Allow Special Instruction for item** (item)
6. Hit **Save** button
