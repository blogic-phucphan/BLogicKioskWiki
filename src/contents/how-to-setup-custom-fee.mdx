# Setup Custom Fee For Kiosk

To setup custom fee for kiosk

1. Go to the license web.
2. Choose the **Store** tab.
3. Select your store.
4. Click on the **Credit Card** tab.
5. Choose Cash Discount Program
6. choose the **Kiosk** section.
5. Enter a value in the **Max base amount** field and enter custom fee.
6. Click the **Save** button.

<Blockquote type="warning">
   To apply a custom fee to a ticket, you must enter a value greater than 0 in the max base amount field.
</Blockquote>

### To Change Custom Fee Name

1. Enter a custom fee name in the field.
2. Click the **Save** button.
