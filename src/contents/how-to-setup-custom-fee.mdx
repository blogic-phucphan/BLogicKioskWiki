# Setup custom fee for kiosk

To setup custom fee for kiosk

1. Go to license web 
2. Choose **Store** tab
3. Choose your store
4. Hit **Credit Card** tab, choose **Kiosk** section
5. Enter **Max base amount** field
6. Hit **Save** button

<Blockquote type="warning">
   To apply a custom fee to a ticket, you must enter max base amount field greater than 0
</Blockquote>

### To change custom fee name

1. Enter custom fee name field
2. Hit **Save** button
