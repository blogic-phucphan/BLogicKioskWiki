# View all tickets

To view all tickets

1. Go to **Setting** (Click 5 times on the footer in the start screen)
2. Hit **Batch manager**

Successfully paid tickets will be displayed in this table.


<img src="/assets/setup-kiosk/7b07ab1591a808566a2528c1d20d541e.png" alt="" />

### Settle batch manual

To settle batch manual: Hit **Settle this batch**

# View detail ticket
